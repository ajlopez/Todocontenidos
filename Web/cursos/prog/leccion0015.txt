Cerremos el formulario, y programemos algo para el botón. Haciendo doble click sobre el mismo, aparecerá

.im

de código, donde podemos ingresar

.code
Option Explicit

Private Sub Command1_Click()
    Dim nombre as String   ' declaramos la variable

    nombre = Text1.Text ' tomamos la propiedad Text del control Texto1

    MsgBox "Bienvenido " & nombre  ' ponemos una ventana de mensaje en pantalla
End Sub
.

Observe que usamos {{Option Explicit}}. Por primera vez, accedemos via código a una propiedad de un control.
Cada control que tengamos en nuestro formulario, es para nosotros una variable que podemos usar en todas
las rutinas del formulario. Como variable, tiene como nombre el nombre del control. El nombre de todo control
se asigna automáticamente, pero nosotros podemos cambiarlo, cambiando su propiedad {{name}}.

Ahora, como variable, no tiene un único valor, sino que tiene varias propiedades. Para acceder a una propiedad
determinada del control, ponemos punto {{.}} y el nombre de la propiedad. Así, {{Text1.Text}} es el valor
de la propiedad {{Text}} del control {{Text1}}.

Una función nueva (que había sido mencionada en un ejercicio anterior) es {{MsgBox}}. Nos muestra una ventana
de mensaje con el texto que le pongamos como parámetro. Hasta que no cerramos la ventana, no volvemos a nuestro
formulario:
